## Primary user experience goals
- **Calm, beautiful UI** that still prioritizes **readability**.

- **Encouraging feedback loops** (points, completion, streaks).

## UX constraints / guardrails
- Glass/gradient effects should never reduce text contrast below comfortable levels.
- Themed/pastel palettes are allowed, but key content needs a readable backing (scrim, card, stronger blur, or adaptive text color).
