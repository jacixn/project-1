# Character Image URL Reference

This file shows the imageUrl pattern for every character in your app. When you add a new character to `bible-characters.json`, use these URLs.

## Base URL Pattern
```
https://raw.githubusercontent.com/jacixn/biblely-character-data/main/profileImage/[CHARACTER_NAME].png
```

## Image Naming Rules

1. **Lowercase**: All names in lowercase (e.g., `moses.png`, not `Moses.png`)
2. **Hyphens for spaces**: Use hyphens instead of spaces (e.g., `lots-wife.png`)
3. **Apostrophes removed**: Remove apostrophes (e.g., `jobs-wife.png`, not `job's-wife.png`)
4. **Special characters**: For parentheses like "Eliezer (son of Moses)", use descriptive suffix (e.g., `eliezer-moses.png`)
5. **Consistent naming**: If you use `adam1.png`, stick with that pattern for similar characters

## Workflow for Adding a New Character

1. Create/get the character image (PNG format recommended)
2. Name it following the rules above
3. Upload to GitHub: https://github.com/jacixn/biblely-character-data/tree/main/profileImage
4. Edit `bible-characters.json` on GitHub
5. Add the character entry with the imageUrl from this list
6. Commit changes
7. App updates automatically!

---

**Pro Tip**: You can copy-paste these URLs directly when adding new characters. Just make sure you've uploaded the corresponding image file to GitHub first!
